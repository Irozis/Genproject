Genproject-demo: запуск portable-версии

1. Откройте папку Genproject-demo.
2. Запустите Genproject Demo.exe.
3. На стартовом экране нажмите "Импортировать .json".
4. Выберите файл demo-data/demo-project.json.
5. Для показа генерации можно также создать новый проект и выбрать изображение demo-data/images/product-square.svg.
6. В форматах выберите "Выбрать рекомендованные" или оставьте форматы из импортированного проекта.
7. На шаге экспорта используйте:
   - PNG ZIP для выгрузки PNG;
   - SVG ZIP для выгрузки SVG;
   - JSON проект для сохранения проекта.

Для переноса на флешку скопируйте всю папку Genproject-demo целиком. Node.js, npm, исходный код и интернет не нужны.

Резервный показ: готовые PNG лежат в demo-data/exports.
