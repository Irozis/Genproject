Genproject demo-data

Содержимое папки:
- demo-project.json - готовый проект для импорта в приложение.
- source-material.txt - исходный рекламный материал для ручного ввода.
- images/product-square.svg - основное изображение товара.
- images/product-horizontal.svg и images/product-vertical.svg - дополнительные варианты изображения.
- exports/*.png - готовые PNG для резервного показа.

Быстрый сценарий демонстрации:
1. Запустите приложение Genproject.
2. На стартовом экране нажмите "Импортировать .json".
3. Выберите файл demo-data/demo-project.json.
4. Проверьте выбранные форматы: VK square, Instagram story, Yandex Market card, Ozon card, Avito listing.
5. При необходимости нажмите "Выбрать рекомендованные" или измените форматы вручную.
6. Перейдите к шагу предпросмотра/экспорта.
7. Для PNG нажмите экспорт PNG ZIP.
8. Для SVG нажмите экспорт SVG ZIP.
9. Для JSON выберите экспорт JSON проекта.

Если приложение не запускается на компьютере защиты, откройте изображения из папки demo-data/exports.
