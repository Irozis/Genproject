# Ad Format Research

Дата проверки: 2026-06-01.

## Добавленные площадки

В каталог `src/data/adFormats.ts` добавлены и нормализованы форматы российских рекламных площадок: Яндекс Директ / РСЯ, VK Реклама / ВКонтакте, Одноклассники / myTarget-подобные социальные размещения, Авито Реклама, Ozon Performance, Wildberries / WB Media, 2ГИС, Telegram Ads и практические Telegram channel post форматы.

Старые пресеты приложения сохранены как ключи совместимости: `vk-square`, `vk-vertical`, `vk-landscape`, `vk-stories`, `telegram-story`, `instagram-story`, `wb-card`, `wb-infographic`, `ozon-card`, `ozon-fresh-square`, `yandex-market-*`, `avito-*`, `yandex-rsy-*`.

## Что внесено

Каталог хранит размеры холста, площадку, группу размещения, устройство, цель, поддерживаемые типы файлов, лимит веса, HTML5/SVG/PNG/JPEG-флаги, safe area, visible area, overlay zones, текстовые лимиты, правила логотипов, правила модерации, источник, дату проверки и confidence.

Для Яндекс РСЯ добавлена серия графических и медийных баннеров 160x600, 240x400, 240x600, 300x250, 300x300, 300x500, 300x600, 320x50, 320x100, 320x480, 336x280, 480x320, 728x90, 970x250, 1000x120, включая 2x и 3x high-density варианты.

Для VK, OK, Авито, Ozon, WB, 2ГИС и Telegram добавлены размеры из продуктового задания, включая маркетплейс-баннеры, social post форматы, сторис, карты и HTML5-размещения.

## Учитываемые ограничения

Приложение использует размеры формата, safe area, visible area, overlay zones, required padding, min font size, text limits, file type support и приблизительный max file size при генерации, предпросмотре, валидации и экспорте.

В предпросмотре можно включить безопасные зоны, зоны маркировки, видимую область и границы обрезки. Валидатор предупреждает, если важный текст, CTA или логотип выходят из safe/visible area либо попадают в зоны маркировки Ozon/WB/других площадок.

## Приблизительные данные

Часть требований внесена как `industry_reference` или `manual` с confidence `medium`/`low`: лимиты Ozon 150/120 КБ, некоторые WB visible area, некоторые 2ГИС текстовые лимиты и практические Telegram post размеры. Эти ограничения нужно вручную сверять с актуальными кабинетами площадок перед запуском кампании.

Приложение не заменяет финальную модерацию площадки. Оно помогает подготовить макет технически корректнее и заранее увидеть типовые риски.
